Siesta PHP
Stored procedure based object relational mapper for php

Overview

Siesta is an object relational mapper based on stored procedures for PHP.
There are many good orm for php available. So why write a new one?

  • First of all : it is fun
  • Stored procedures are fast, that is why every interaction with the database is done via Stored procedures

Installation

Installation with composer

composer require gm314/siesta

Create configuration file with

vendor/bin/siesta init
